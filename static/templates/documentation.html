Zeri Art Market Knowledge Graph (ZAMO-KG)

This knowledge graph contains material on the art market, produced within a research project carried out by the Fondazione Federico Zeri (University of Bologna). It is based on the Zeri Art Market Ontology (ZAMO), and originates from source data published in the "Mercato dell'Arte" repository by the Fondazione Federico Zeri.

ZAMO Ontology

The structure of ZAMO-KG is built upon the Zeri Art Market Ontology (ZAMO), an OWL ontology designed for describing agents, events and sources relevant to the art market domain. The ontology enables structured and machine-actionable data, providing a framework for representing art market information.

Documentation for ZAMO can be found here: ZAMO Documentation. The ontology's source code is available on GitHub.

Construction of the Knowledge Graph

The Zeri Art Market Knowledge Graph (ZAMO-KG) is created starting from source tabular data (originally published in the "Mercato dell'Arte" repository by the Fondazione Federico Zeri). Python scripts were used to iterate over the datasets to create a Turtle file and validate it with competency questions. This static website for content negotiation was lastly built with Simple PUblisher of Knowledge (SPUK).
